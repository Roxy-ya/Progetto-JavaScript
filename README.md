# Progetto-JavaScript - Counter App
Si tratta di una applicazione sviluppata in **JavaScript puro**, che implementa un contatore con pulsanti per incrementare, decrementare e resettare il valore.

L'interfaccia viene creata mediante manipolazione del DOM, e il valore salvato tramite localStorage, in modo da rimanere anche dopo il caricamento della pagina


# Comportamento
Il contatore parte dal valore "0".
-'+' incrementa il valore
-'+' decrementa il valore
-'Reset' riporta il valore a "0"


# Funzionalità
-Incremento del contatore
-Decremento del contatore
-Reset del valore
-Salvataggio mediante **localStorage**
-Layout con Css
